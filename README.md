# DuckStation Dependencies Build Scripts

This repository contains scripts to build the dependencies required for DuckStation on various platforms.

## Supported Platforms

- Windows x64
- Windows ARM64
- Linux x64
- Linux ARM64
- Linux ARM32/ARMHF
- MacOS x64
- MacOS ARM64

## Instructions

Please refer to the DuckStation README for information on how to use these scripts.

## Automated Builds

Archives of the dependency packages are automatically built by GitHub actions. Each release is tagged with the commit date to allow old versions of DuckStation to be built.
